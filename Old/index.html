SYMAP

A D3 implementation of the overprinting technique used by early automated cartography platforms to render shaded areas on line printers. The map also makes use of an in-browser interpolation function that generates a contour map from areal data. Displays US Census 2010 population density data by default.

Source code on GitHub.

U.S. Power Plants

A proportional symbol map, coded using Leaflet, that displays all hydroelectric, solar, and coal power plants in the United States scaled by their total power output. Data provided by Rich Donohue.

Source code on GitHub.

Unemployment in Kentucky

A chorpleth map that displays unemployment in Kentucky counties between 2000 and 2013. The map is coded in Leaflet with some D3 tacked on to visualize time-series data.Data provided by Rich Donohue.

Source code on GitHub.

Power Plants Near Ann Arbor

A proportional symbol map, coded using Leaflet, that displays all hydroelectric, solar, and natural gas power plants within 500 kilometers of Ann Arbor, MI. Data provided by Rich Donohue.

Source code on GitHub.

About Me

I am a designer, geographer, academic, and educator whose research examines the use of geospatial technologies within the design disciplines and active role of those disciplines in the making of GIS. I am particularly interested in the ecological design tradition within landscape architecture and the ways in which its view of nature, methods, assumptions, and techniques, are mobilized in GIS software.
